Place your resume PDF at the root of this site with the filename `Lavish_Resume.pdf`.

Path: `Modern-Portfolio/Lavish_Resume.pdf`

When the file is present, the "Download Resume" button on the home (hero) section will trigger a direct download. If you want a different filename, update the `href` in `index.html` accordingly.

To create or replace the resume on Windows PowerShell, you can copy a file like this:

Copy-Item -Path "C:\path\to\YourResume.pdf" -Destination "C:\Users\snsiv\Downloads\Modern-Portfolio\Modern-Portfolio\Lavish_Resume.pdf" -Force
